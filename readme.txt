编译下载方法(注意修改make.sh中的进程数和写入sd的设备路径):
$ make gblw210_config
$ ./make.sh
$ ./flash_sd.sh /dev/sdb

最后插入sd卡,短接启动模式跳线,启动


SD卡数据分布:
address     content     size(byte)  size(human-readable)
0-1ff       mbr         512         512
200-41ff    210.bin     16384	    16K
4200-61ff   env         8192        8K        
6200-3fffff u-boot.bin  4169216     3.97M       
400000+     sd-data


mach type: 21986
